Property letter-spacing dan word-spacing

Dibuat oleh Abel Christian pada 17 Juni 2020

Property letter-spacing

Property ini digunakan untuk membuat jarak antara huruf dengan huruf lainnya

Syntax:

                        
                        p{
                            letter-spacing: 10px;
                        }

Output:

Letter Spacing

Untuk nilai positif, hurufnya akan merenggang. Dan jika nilainya negatif maka hurufnya akan merapat.

Syntax:

                        
                        p{
                            letter-spacing: -2px;
                        }

Output:

Letter Spacing

Perlu diperhatikan penggunaan nilai negatif dapat merapatkan huruf, berkemungkinan huruf/kalimat tidak dapat dibaca.

Penjelasan

Dirasa sudah jelas, letter-spacing ini digunakan untuk merenggangkan (jarak) antar huruf dan nilainya bisa positif atau negatif jika ingin kembali normal maka bisa diisi dengan nilai normal.

Property word-spacing

Property ini dugnakan untuk membuat jarak antar kata

Syntax:

                        
                        p{
                            word-spacing: 20px;
                        }

Output:

Word Spacing

Nilainya bisa juga diisi dengan negatif.

Syntax:

                        
                        p{
                            word-spacing: -10px;
                        }

Output:

Word Spacing

Penjelasan

Kurang lebih sama seperti property letter-spacing untuk nilainya namun perbedaanya property word-spacing ini adalah untuk membuat jarak antar kata.

Ya mungkin itu saja yang bisa ditulis tentang property letter-spacing dan word-spacing pada CSS, ini hanya sekadar dokumentasi pembelajaran mandiri kurang lebih Terima Kasih.
